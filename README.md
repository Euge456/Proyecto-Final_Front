# 💍 AURA Joyería - Proyecto Final Front-End

## Descripción

AURA Joyería es un e-commerce desarrollado como proyecto final de Front-End. El sitio está especializado en la venta de anillos y ofrece una experiencia de compra moderna, elegante e intuitiva.

El proyecto fue desarrollado utilizando HTML5, CSS3 y JavaScript, incorporando Bootstrap para lograr un diseño responsive adaptable a distintos dispositivos.

---

## Funcionalidades

* Página de inicio con presentación de la marca.
* Catálogo exclusivo de anillos.
* Página completa de colección con 20 modelos.
* Carrito de compras dinámico.
* Agregar productos al carrito.
* Modificar cantidades.
* Eliminar productos del carrito.
* Cálculo automático del total.
* Persistencia del carrito mediante LocalStorage.
* Formulario de contacto.
* Diseño responsive para computadoras, tablets y celulares.

---

## Características del diseño

* Estilo minimalista y elegante.
* Paleta de colores en tonos blanco, negro y dorado.
* Tipografía Playfair Display para títulos.
* Tipografía Montserrat para el contenido.
* Animaciones suaves en botones y tarjetas.
* Navegación intuitiva.

---

## Funcionalidades JavaScript

* Renderizado dinámico de productos.
* Gestión completa del carrito.
* Actualización automática del total.
* Formato de precios en pesos argentinos.
* Almacenamiento del carrito en LocalStorage.
