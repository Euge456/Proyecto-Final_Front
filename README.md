# Aura Joyería - Proyecto Final E-Commerce
Este proyecto es una tienda online de joyería fina e interactiva.
Consume datos en tiempo real de FakeStoreAPI filtrando por la categoría 'jewelery'.

### Tecnologías utilizadas:
- HTML5 Semántico
- CSS3 (Flexbox, Grid y Media Queries)
- Bootstrap 5
- JavaScript Vanilla (Fetch API, LocalStorage y manipulación del DOM)
